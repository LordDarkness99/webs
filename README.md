# 3D Developer Portfolio

A modern and interactive 3D portfolio website built with React and Three.js.
This project showcases my skills, projects, and experience in a clean and engaging way.

---

## 🚀 About

This portfolio combines responsive design, smooth animations, and 3D elements
to create a unique user experience.

---

## 🛠 Tech Stack

- React.js
- React Router
- Three.js
- React Three Fiber
- Framer Motion
- Tailwind CSS
- Vite
- EmailJS

---

## ✨ Features

- Interactive 3D Hero Section
- Smooth Animations
- Skills & Experience Timeline
- Project Showcase
- Contact Form
- Fully Responsive Design

---

## 📦 Installation

Clone the repository:

git clone [https://github.com/AimTheSun/portofolio-website-3d.git](https://github.com/LordDarkness99/webs.git)

Navigate to the project folder:

cd portofolio-website-3d

Install dependencies:

npm install

Run development server:

npm run dev

Open the local URL provided by Vite in your browser.
